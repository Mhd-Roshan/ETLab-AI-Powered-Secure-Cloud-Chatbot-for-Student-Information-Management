## MCA (Master of Computer Applications) - 2 Years Program

### Program Overview
- **Duration**: 2 Years (4 Semesters)
- **Total Credits**: 90 Credits
- **Eligibility**: Bachelor's degree in any discipline with Mathematics at 10+2 level
- **Intake**: 60 Students
- **Evaluation**: 50% CIE + 50% SEE

### MCA Semester 1

#### Core Subjects
1. **Mathematical Foundations for Computing (MCA101)**
   - Credits: 4
   - Topics: Discrete Mathematics, Linear Algebra, Probability, Statistics
   - Activities: Problem solving sessions, Mathematical modeling

2. **Advanced Data Structures (MCA102)**
   - Credits: 4
   - Topics: Trees, Graphs, Hashing, Advanced sorting algorithms
   - Activities: Coding assignments, Algorithm analysis

3. **Computer Organization and Architecture (MCA103)**
   - Credits: 3
   - Topics: CPU design, Memory hierarchy, I/O systems
   - Activities: Simulation projects, Hardware labs

4. **Object-Oriented Programming using Java (MCA104)**
   - Credits: 4
   - Topics: OOP concepts, Java programming, Exception handling
   - Activities: Java projects, Code reviews

5. **Database Management Systems (MCA105)**
   - Credits: 4
   - Topics: SQL, Normalization, Transaction management
   - Activities: Database design projects, Query optimization

#### Lab Courses
6. **Data Structures Lab (MCA106)**
   - Credits: 2
   - Activities: Implementation of data structures in C/C++

7. **Java Programming Lab (MCA107)**
   - Credits: 2
   - Activities: Java application development

8. **DBMS Lab (MCA108)**
   - Credits: 2
   - Activities: SQL queries, Database design

**Semester 1 Total Credits**: 25

## Evaluation System (KTU Pattern)

### Continuous Internal Evaluation (CIE) - 50 Marks
1. **Assignments**: 10 marks
2. **Tests** (2 tests, best of 2): 30 marks
3. **Attendance**: 5 marks
4. **Seminar/Viva**: 5 marks

### Semester End Examination (SEE) - 50 Marks
- **Duration**: 3 hours
- **Pattern**: Theory + Problem solving
- **Pass Marks**: Minimum 50% overall (40% in SEE mandatory)

### Grading System
- **S**: 10 (90-100%)
- **A+**: 9 (85-89%)
- **A**: 8.5 (80-84%)
- **B+**: 8 (75-79%)
- **B**: 7 (70-74%)
- **C**: 6 (60-69%)
- **P**: 5 (50-59%)
- **F**: 0 (Below 50%)

### Attendance Requirements
- **Minimum**: 75% mandatory
- **Condonation**: Up to 10% with medical certificate
- **Below 65%**: Not eligible for university exams
